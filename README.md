# 🔥 AI GitHub Roast

AI GitHub Roast is a fun web application that analyzes GitHub profiles and generates hilarious roasts based on repositories, coding activity, languages used, and profile information.

Whether you're a coding genius or a commit-once-a-month developer, this app is ready to roast your GitHub profile in **English** or **Malayalam**. 😈

## 🚀 Live Demo

https://ai-github-roast-lpot3vkx9-saf369s-projects.vercel.app

## ✨ Features

- 🔍 Analyze any public GitHub profile
- 😂 Generate funny AI-powered roasts
- 🌍 Supports:
  - English Roasts
  - Malayalam Roasts
- 📊 Uses GitHub profile and repository data
- ⚡ Fast and simple interface
- 📱 Responsive design

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- GitHub API
- AI-powered roast generation

## 🎯 How It Works

1. Enter a GitHub username.
2. Select your preferred roast language.
3. The app fetches GitHub profile information.
4. AI analyzes repositories, languages, and activity.
5. Receive a personalized roast instantly.

## 💡 Example

### English Roast

> You have 47 repositories, but half of them are "final-final-v2-actually-final". Your commit history looks like a New Year's resolution that lasted three days.

### Malayalam Roast

> GitHub profile kandal thonnum valiya developer aanennu, pakse repositories open cheythal README polum illa. Commit history kandal semester examinu munpu mathram padikkunna student pole undu.
